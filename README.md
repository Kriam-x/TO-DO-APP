# To-Do JS App

## Overview
A simple JavaScript To-Do application with persistent state using localStorage. Allows adding, completing, and deleting tasks in real-time.

## Features
- Add, delete, and mark tasks as completed
- Uses DOM manipulation and event listeners
- Persistent storage using `localStorage`

## Tech Stack
- JavaScript
- HTML, CSS
- DOM API

## Usage
1. Open `index.html` in a browser
2. Add tasks in the input box and click "Add"
3. Click tasks to toggle completion, or delete with the button
